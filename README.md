# ZdenekCulik/tap

Homebrew tap for my macOS apps.

## Whisper Pro

Native macOS voice-to-text: press a hotkey, speak, the text is typed in wherever your cursor is.

```bash
brew install --cask zdenekculik/tap/whisper-pro
```

Source and docs: https://github.com/ZdenekCulik/whisper-pro

Not a fan of the terminal? Grab the signed `.dmg` from
[Releases](https://github.com/ZdenekCulik/whisper-pro/releases/latest).
